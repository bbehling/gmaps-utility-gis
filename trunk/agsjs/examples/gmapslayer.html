Use Google Maps in ESRI ArcGIS JSAPI Documentation
  • Click buttons to switch base map between ESRI and Google. Note the "Gray", "Night" and "Random" styles. The "random" will show differently each time clicked.
  • Drag pegman icon on top left corner to map to see street view(hold mouse for preview).
  • Check layer on/off to customize base map. Note not all sub layer visible at all scales.
