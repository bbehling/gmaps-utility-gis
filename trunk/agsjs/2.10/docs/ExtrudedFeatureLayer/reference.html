Extruded Feature Layer

A canvas based ESRI ArcGIS Server JavaScript API ported from Open Streem Map Buildings.(osmbuildings.org).

For a description and examples of how to use this library, check out the how-to.

class ExtrudedFeatureLayer

This class is a ExtrudedFeatureLayer, such as a polygon feature layer with a height attribute.

Constructor

Constructor Description
ExtrudedFeatureLayer(url:string||FeatureCollection, opts:ExtrudedFeatureLayerOptions) Create an ExtrudedFeatureLayer

class ExtrudedFeatureLayerOptions

This is an object literal that specify the options for each ExtrudedFeatureLayer. There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
defaultHeight number optional. default Height to use if the height value is 0. default=0;
extentScaleRatio number optional. extra buffer on map extent to load features to reduce server traffic. default=1.5;
heightAttribute string required. name of the attribute for height;
heightScaleRatio number optional. number used to multiple the value from service. default=1;
mode int optional. agsjs.layers.ExtrudedFeatureLayer.MODE_ONDEMAND | MODE_SNAPSHOT. default=ON_DEMAND
query esri.tasks.Query optional. query set on the feature layer for retrieving data.
style Object object with color, roofColor (#ffcc00' 'rgb(255,200,200)' 'rgba(255,200,200,0.9)), strokeRoofs (boolean)
