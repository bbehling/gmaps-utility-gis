MaxInfoWindow

This lib provides a similar infowindow behavior avaiable at maps.google.com. The maxContent of info window contains multiple tabs.

For a description and examples of how to use this library, check out the how-to.

class GMap2

This is new methods added to Google Maps API's GMap2 class.

Methods

Methods Return Value Description
openInfoWindowMaxTabs(latlng:GLatLng, node:Node, summary:Node, tabs:GInfoWindowTab[], opts:MaxInfoWindowOptions) None Opens an info window with max content at the given point. The content of the info window is given as DOM node. The max content has summary info and a set of tabs. The content of each tab is DOM node.
openInfoWindowMaxTabsHtml(latlng:GLatLng, html:String, summary:String, tabs:GInfoWindowTab[], opts:MaxInfoWindowOptions) None Opens an info window with max content at the given point. The content of the info window is given as HTML text. The max content has summary info and a set of tabs. The content of each tab is html.
