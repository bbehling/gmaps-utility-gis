Key Drag Zoom

This lib provides a very simple drag zoom. Holding a user-defined special key (shift | ctrl | alt) while dragging a box will zoom to the desired area. Only one line of code GMap2.enableKeyDragZoom() is needed.

For a description and examples of how to use this library, check out the how-to.

class DragZoom

This class represents a drag zoom object for a map. It can be activated by holding a special key. This is no constructor for this class. Use GMap2.getDragZoomObject() to gain access to this object.

Events

Events Arguments Description
activate None This event is fired after the DragZoom tool is activated.
deactivate None This event is fired while the user release the key
drag southwestPixel:GPoint, northeastPixel:GPoint This event is repeatedly fired while the user drags the box. The southwest and northeast point are passed as parameter of type GPoint (for performance reasons), relative to map container. Note: the event listener is responsible for converting Pixel to LatLng if necessary, using GMap2.fromContainerPixelToLatLng.
dragend newBounds:GLatLngBounds This event is fired after drag end. Note that the dragend event is not fired if the hot key is released before the end of the drag operation.
dragstart startLatLng:GLatLng This event is fired when drag started.

class GMap2

These are new methods added to Google Maps API's GMap2 class.

Methods

Methods Return Value Description
disableKeyDragZoom() None Disable drag zoom
enableKeyDragZoom(opt_zoomOpts:KeyDragZoomOptions) None Enable drag zoom. User can zoom to a point of interest by holding a special key (shift | ctrl | alt ) while dragging a box.
getDragZoomObject() DragZoom Returns the DragZoom Object on this map instance after GMap2.enableKeyDragZoom is called. Event Listeners can be attached afterwards.
keyDragZoomEnabled() Boolean Returns true if the drag zoom is enabled.

class KeyDragZoomOptions

This class represents the optional parameter passed into GMap2.enableDragBoxZoom(). There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
boxStyle Object the css style of the zoom box. e.g. {border: '2px dashed red'}
key String the modifier key to use while this.dragging_ the box, shift | alt | ctrl . Default is shift.
paneStyle Object the css style of the pane which overlays the this.map_ when a drag zoom is activated. e.g. {backgroundColor: 'gray', opacity: 0.2}.
