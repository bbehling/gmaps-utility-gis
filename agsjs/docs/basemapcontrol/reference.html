Base Map Groups widget for ArcGIS Server JavaScript API

Base Map Groups widget. Typically placed on top right corner of the map, and organize base maps into groups that displayed as Tabs. Can use Google maps, and can have slider for images.

For a description and examples of how to use this library, check out the how-to.
